# Kibum Moon - Personal Website

This repository contains the source code for my personal website.

## Tech Stack
- React
- Vite
- TypeScript
- Tailwind CSS (if applicable)

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment
The website is automatically deployed to GitHub Pages using `npm run deploy`.
